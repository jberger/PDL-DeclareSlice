=head1 NAME

PDL::DeclareSlice - An alternative to L<PDL::NiceSlice> using L<Devel::Declare>.

=head1 SYNOPSIS

 use PDL
 use PDL::DeclareSlice;

 my $pdl = xvals(5);
 print sl $pdl(1:2); # prints [1 2]
